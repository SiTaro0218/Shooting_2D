## **2\. コアゲームデザイン (Core Game Design)**

### **2.1. ゲームの核となる体験 (Core Loop)**

プレイヤーは自機を操作し，自動スクロールするマップを進みます．対空ショットと対地ボムの二種類の攻撃を使い分け，空中敵と地上物を破壊し，ハイスコアを目指します．ステージは毎回異なる構成で生成され，敵の動きも学習的に変化します．

### **2.2. 主要なルールとシステム**

| 項目 | 内容 |
| :---- | :---- |
| **操作系** | キーボード（WASDまたは矢印キー）またはゲームパッドによる移動，対空ショット，対地ボム． |
| **攻撃** | **対空ショット** (連射可能)，**対地ボム** (クールダウン/残弾制，地上物専用)． |
| **ライフ** | シールド制を採用．シールドがゼロになるとゲームオーバー． |
| **スコア** | 敵の撃破，特定の地上物の破壊，生存時間に応じて加算． |
| **ゲームサイクル** | スタート → 自動生成ステージ進行 → 被弾/時間切れでゲームオーバー → スコア記録・リスタート． |

### **2.3. 特徴的なシステム**

#### **A. プロシージャル・ステージ生成 (Procedural Generation)**

* **目的:** ステージの再利用性を高め，毎回異なるプレイ体験を提供．  
* **手法:** シンプルなパーリンノイズやランダムアルゴリズムに基づき，地面のテクスチャや壁/水面などの障害物を自動配置します．  
* **構造:** 画面外に出たマップチャンクを破棄し，上部で新しいチャンクを生成・接続する「リサイクリング」方式で実装．

#### **B. 簡易 AI 制御と進化 (Adaptive AI)**

* **初期 AI:** 敵の行動は**有限状態機械 (FSM)** で制御します（例: 巡回→プレイヤー追跡→攻撃）．  
* **進化要素 (目標):** プレイヤーの行動データ（例：特定のマップ区間での生存時間，回避率）を記録．次回のプレイ時に，敵の**初期配置や巡回速度，弾の発射頻度**などに**ランダムな調整を加える**ことで，難易度やパターンを間接的に進化させます．

| 優先度 | 項目 | 概要 | 担当スクリプト (想定) |
| :---- | :---- | :---- | :---- |
| **A (高)** | **A-1. UI 実装** | スコア，ライフ，ゲームオーバー/クリア画面，リスタートボタンの表示． | GameManager，UI 制御スクリプト |
| **A (高)** | **A-2. マップ制御** | 一定速度の自動縦スクロール，マップチップの生成・破棄サイクル． | MapGenerator，GameManager |
| **A (高)** | **A-3. 自機制御** | プレイヤー機の移動（水平・垂直），対空ショットの発射ロジック． | PlayerController，BulletController |
| **A (高)** | **A-4. 衝突処理** | 敵・弾・地形の当たり判定と，OnTriggerEnter2Dによる処理． | PlayerController，ItemData |
| **B (中)** | **B-1. 敵の FSM** | 敵機の基本行動パターン（待機，移動，攻撃）の実装． | EnemyController (FSM) |
| **B (中)** | **B-2. 対地ボム** | ボムの投下アニメーション，着弾判定，地上物の破壊処理． | PlayerController，BombController |
| **B (中)** | **B-3. パワーアップ** | アイテムのドロップ，取得，自機ショットの強化（連射数，弾速）． | ItemData，PlayerController |

| **C (低)** | **C-2. AI フィードバック** | 収集データに基づき，次のステージの敵配置を調整するロジック． | MapGenerator，EnemySpawner |
| **C (低)** | **C-3. サウンド** | BGM，効果音の組み込みと音量制御． | SoundManager |

## **4\. アセット要件 (Asset Requirements)**

* **グラフィック**: 2D ドット絵フリー素材を使用．  
  * **重要設定:** Filter Mode を **Point (No Filter)** に設定．  
* **自機**: 1セット（移動，被弾，ショット，爆発アニメーション）．  
* **敵機**: 空中敵 3種，地上物 2種．  
* **UI**: 日本語対応フォント (TextMeshPro用フォントアセットを生成済みとする)．  
* **サウンド**: BGM，効果音（ショット，爆発，被弾，アイテム取得）．
